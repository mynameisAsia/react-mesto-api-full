# react-mesto-api-full
Приложение проекта `Mesto`, включающий фронтенд и бэкенд части приложения со следующими возможностями: авторизации и регистрации пользователей, операции с карточками и пользователями. Бэкенд расположите в директории `backend/`, а фронтенд - в `frontend/`. 
  
Ссылка на сайт: http://amam.mesto.nomoredomains.rocks
